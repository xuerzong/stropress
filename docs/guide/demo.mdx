# Markdown Demo

This page intentionally collects common Markdown and MDX patterns so you can inspect how the docs theme renders typography, spacing, blocks, tables, and interactive content.

## Typography

Plain text should sit comfortably next to **bold text**, _italic text_, **_bold italic text_**, ~~strikethrough~~, `inline code`, and <kbd>Ctrl</kbd> + <kbd>K</kbd> style keyboard hints.

You can also verify [internal links](/guide/getting-started) and [external links](https://astro.build).
